## 二、安卓兼容性考量

### 2.1 建议支持的 API Level

| 等级 | API Level | 市场份额 | 建议 |
|------|-----------|----------|------|
| 最低支持 | API 24 (Android 7.0) | ~95% 覆盖率 | **MVP 最低版本** |
| 推荐目标 | API 28+ (Android 9.0+) | ~85% 覆盖率 | 完整功能支持 |
| 最新支持 | API 34 (Android 14) | 新特性测试 | 向后兼容验证 |

### 2.2 设备兼容性测试矩阵

| 测试项 | 低端机 | 中端机 | 高端机 |
|--------|--------|--------|--------|
| 示例机型 | 红米 9A (3GB) | 小米 12 (8GB) | Pixel 8 (12GB) |
| API Level | 24-26 | 28-30 | 33-34 |
| 屏幕分辨率 | 720p | 1080p | 1440p |
| 启动时间 | <3s | <2s | <1.5s |
| 卡片动画 | 无掉帧 | 流畅 60fps | 流畅 60fps |
| 内存占用 | <150MB | <200MB | <250MB |

### 2.3 屏幕适配测试

- [ ] 小屏幕 (5.0寸以下，如 Pixel 4a)
- [ ] 标准屏 (5.0-6.5寸，主流机型)
- [ ] 大屏/折叠屏 (6.5寸以上，需验证布局适配)
- [ ] 横竖屏切换 (卡片布局是否正确适配)

### 2.4 关键兼容性检查点

| 检查项 | 说明 |
|--------|------|
| 存储权限 | Android 10+ 分区存储适配 |
| 音频焦点 | 与其他音频 App 切换时的行为 |
| 后台恢复 | App 后台长时间后返回，状态不丢失 |
| 深色模式 | 跟随系统主题，界面可读 |
| RTL 布局 | 虽不必须，但需确保不会错乱 |

## 三、用户体验检查点

### 3.1 性能指标

| 指标 | MVP 目标 | 测试方法 |
|------|----------|----------|
| 冷启动时间 | ≤ 2.5 秒 | 从点击图标到首页可交互 |
| 页面切换 | ≤ 300ms | 点击导航到页面渲染完成 |
| 卡片滑动 | 60 FPS | Systrace/GFX 检测 |
| 音频加载 | ≤ 500ms | 点击发音到声音播放 |
| APK 大小 | ≤ 50MB | 含发音文件的完整包 |

### 3.2 交互流畅度检查

| 检查项 | 通过标准 |
|--------|----------|
| 卡片滑动 | 无卡顿，边界反弹自然 |
| 按钮反馈 | 点击有视觉反馈(涟漪/变色)，响应 ≤ 100ms |
| 列表滚动 | 假名/单词列表快速滚动无掉帧 |
| 转场动画 | 页面跳转动画完整，无撕裂 |
| 加载状态 | 数据加载时有 loading 提示，不白屏 |

### 3.3 可用性检查

| 检查项 | 通过标准 |
|--------|----------|
| 空状态 | 无数据/无网络时有友好提示 |
| 错误提示 | 操作失败时有明确中文提示 |
| 可访问性 | 按钮大小 ≥ 48dp，文字对比度 ≥ 4.5:1 |
| 手势冲突 | 卡片滑动与导航不冲突 |

### 3.4 离线体验

| 检查项 | 通过标准 |
|--------|----------|
| 离线启动 | 无网络时 App 正常启动 |
| 离线学习 | 已加载的假名/单词可正常浏览、测验 |
| 离线提示 | 尝试联网功能时有明确提示 |

## 四、测试执行计划

### Phase 1: 冒烟测试 (开发阶段，每 build)
- TC-H01, TC-H02 (假名展示)
- TC-Q01 (基础测验)
- TC-D01, TC-D04 (数据完整)

### Phase 2: 功能测试 (Feature Complete 后)
- 全部 P0、P1 用例
- 中端机兼容性验证

### Phase 3: 回归测试 (Release Candidate)
- 全用例回归
- 低端机 + 高端机验证
- 性能指标验收

## 五、Bug 严重等级定义

| 等级 | 定义 | 示例 |
|------|------|------|
| P0-阻塞 | 核心功能不可用，阻塞测试 | App 崩溃、假名不显示、测验无法提交 |
| P1-严重 | 功能缺陷，影响用户体验 | 答对判定为错、发音不播放、严重卡顿 |
| P2-一般 | 非核心功能问题 | UI 错位、提示文字错误、连击显示异常 |
| P3-轻微 | 优化建议 | 动画不够流畅、颜色微调 |
